# robbysailing.com

Robby Meek's LA2028 Olympic Sailing Campaign website.

Built with React + Vite.

## Setup

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`

## Build & Deploy

```bash
npm run build
```

Output goes to `dist/` — deploy to Vercel, Netlify, or GitHub Pages.

## Project Structure

```
src/
  components/   Nav, Footer, Marquee (shared)
  hooks/        useCountdown
  pages/        Home, Biography, EventCalendar, Team, Contact
  App.jsx       Router
  main.jsx      Entry point
  index.css     Global styles
```
